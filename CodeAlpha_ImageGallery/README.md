# Image Gallery

This project was developed as part of my Frontend Development Internship.

A fully responsive and interactive Image Gallery built using HTML, CSS, and JavaScript.

---

## Features

- Responsive Grid Layout
- Modern Glassmorphism UI
- Lightbox View
- Next / Previous Navigation
- Category-Based Image Filtering
- Filter-Aware Navigation Logic
- Hover Effects & Smooth Transitions
- Keyboard Navigation Support
- Active Button Highlighting

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Learning Outcomes

- DOM Manipulation
- Event Handling
- Responsive Design Techniques
- UI/UX Enhancements
- State Management in JavaScript
